# Java To-Do List Application

## Description

This is a simple command-line based To-Do List application developed using Java.  
It allows users to add, view, and delete tasks. Tasks are stored in a text file so they are saved even after restarting the program.

## Features

- Add new tasks
- View all tasks
- Delete tasks
- Persistent storage using file handling
- Uses Object-Oriented Programming (Task class)

## Technologies Used

- Java
- ArrayList (Collections Framework)
- File Handling (FileReader, FileWriter)
- OOP Concepts (Class, Object, Encapsulation)
